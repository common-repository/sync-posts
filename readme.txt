== Description ==

Sync Posts is a WordPress plugin that allows you to sync posts from another website using a URL. With this plugin, you can easily import Post Title, Content, Content Images, Post Tags, Categories, and Featured Images from any WordPress site to your current WordPress site.


== Frequently Asked Questions ==

= How can I sync posts from another website? =

To sync posts from another website, go to the 'Sync Posts' menu in the WordPress admin area, enter the website URL, post status, number of posts, and author, and click the 'Start Syncing' button. The plugin will handle the rest and import the posts for you.

= What data can I import using this plugin? =

You can import Post Title, Content, Content Images, Post Tags, Categories, and Featured Images from any WordPress site to your current WordPress site.

= Can I choose which posts to import? =

Currently, the plugin imports all posts based on the specified criteria (website URL, post status, number of posts). There is no option to select specific posts for import.

= Is it possible to sync posts from a non-WordPress website? =

No, this plugin is designed specifically for syncing posts between WordPress sites. It does not support syncing from non-WordPress websites.

== Installation ==

1. Upload the `sync-posts` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Sync Posts' menu in the WordPress admin area.
4. Enter the required information, such as the website URL, post status, number of posts, and author.
5. Click the 'Start Syncing' button to initiate the sync process.
